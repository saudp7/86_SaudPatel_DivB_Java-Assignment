# Assignment 4 - EJB and JPA

## Folder Contents
- `Ass-4_P01/` - JMS Queue + asynchronous business logic + XML response + client-side XML parsing
- `Ass-4_P02/` - Currency converter using EJB with conversion factors in DB
- `Ass-4_P03/` - Minimal computer store shopping cart using EJB + JPA
- `Ass-4_P04/` - Student and Subject many-to-many data operations with reports
- `Ass-4_P05/` - Shopping mall data-intensive app with Session Facade and validation
- `DB Script.txt` - common SQL script for all Assignment 4 projects

## Project Style
- Same naming convention as previous assignments (`Ass-4_P0x`)
- NetBeans-friendly Maven web projects (`pom.xml`, `nb-configuration.xml`)
- Simple and clean JSP UI (minimal forms, no heavy frontend)

## Run Notes
1. Run `DB Script.txt` in MySQL.
2. Configure your server datasource as `jdbc/ass4DS`.
3. Open each project in NetBeans and deploy to Jakarta EE compatible server.
