# Assignment 1 - Core Java

## Folder Contents
- `Ass-1_P01/` - Inheritance, polymorphism, upcasting, downcasting (`Shape`, `Rectangle`, `Triangle`)
- `Ass-1_P02/` - Swing based address book without JDBC (collection-based storage)
- `Ass-1_P03/` - `DataTable` with `Featurable` interface and `TableException`
- `Ass-1_P04/` - Retail customer GUI using JDBC, CRUD + married customer average income
- `Ass-1_P05/` - Category hierarchy tree printing from `Category_Master`
- `DB Script.txt` - database setup script for JDBC-based tasks

## Mapping to Question Set
1. UML/inheritance task -> `Ass-1_P01`
2. Address Book Swing task -> `Ass-1_P02`
3. DataTable textual interface task -> `Ass-1_P03`
4. Retail customer JDBC GUI task -> `Ass-1_P04`

## Run Notes
- `Ass-1_P01` to `Ass-1_P05` are NetBeans Ant projects (`build.xml` present).
- Open each project in NetBeans and run its main class.
- For JDBC projects, execute `DB Script.txt` first and update DB credentials if required.
